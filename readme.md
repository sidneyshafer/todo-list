## App Usage
Install dependencies
```
npm install
```
Run dev server:
```
npm start
```
App will run on `http://localhost:3000`

Build for production:
```
npm run build
```
Production files will be in the `build` folder.

## Checking for Dependency Updates
```
npm i -g npm-check-updates
ncu -u
npm install
```
